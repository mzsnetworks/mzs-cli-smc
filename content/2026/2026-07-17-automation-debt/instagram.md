Every script you automate is a liability you now own. We only ever count the asset. 💸

Automating a task feels like pure win — did it once, now it runs itself. What nobody logs is what you took on: software that has to be maintained and trusted for life, or it becomes a hazard.

That's automation debt. And it accrues interest ⚙️
→ the playbook that breaks on every vendor API change
→ the script only one person understands
→ the pipeline everyone's afraid to touch
→ the "temporary" automation that's now load-bearing

Manual work degrades gracefully — a human notices when something's off. Unmaintained automation fails silently, at scale, on stale assumptions. The blast radius is bigger than the toil it replaced.

Automation you don't maintain isn't saving time. It's deferring a bill.

Automate less, but own what you automate. A little well-maintained automation beats a graveyard of scripts nobody dares run.
