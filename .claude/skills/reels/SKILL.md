# Reels

Run the **REELS agent**. Read `agents/REELS.md`, `rules/SHARED.md`, `rules/INSTAGRAM.md`, `rules/VOICE.md` (if present), and the source idea (`master.md`), then execute:

1. Turn the idea into a tight script: 30–45 seconds, 2 key points max, hook leading (never opens with "I").
2. Output the **caption alongside the script** plus a single-word comment trigger and visual notes.
3. If the user describes a reference Reel's structure, mirror the shape only — never scrape, download, or copy another creator's words.

Same fact discipline: any stat spoken on camera must trace to a source. Keep caption and script in sync.
