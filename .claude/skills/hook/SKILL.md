# Hook

Run the **HOOK agent**. Read `agents/HOOK.md`, `rules/SHARED.md`, and `rules/VOICE.md` (if it exists), then execute:

1. Produce 6 hook options across the proven shapes (Tension, Stat-led, Contrarian, Confession, Pattern, Reframe), in the author's voice.
2. Mark which hooks lean on a stat (so Factcheck can confirm the number exists).
3. Recommend the strongest one and say why it earns the second line.

Reject clickbait — no fake curiosity gaps, no rage-bait, no unsourced stats in a hook. Hand the pick to `/post` or the Writer.
