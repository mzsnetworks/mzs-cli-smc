# Build Voice

Run the **VOICE agent**. Read `agents/VOICE.md` and `rules/SHARED.md`, then execute it exactly:

1. Run the six-question interview (background, defended opinions, war stories, respected writers, voice fingerprint, content pillars).
2. If the user pasted 3–5 sample posts, analyze them for measurable patterns and fold those in.
3. Write `rules/VOICE.md` in the format defined in `agents/VOICE.md`.

This profile is mandatory for every writing agent once it exists. If `rules/VOICE.md` already exists, update it rather than duplicating. Confirm what changed when done.
